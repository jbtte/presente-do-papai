# 👟 O Tênis do Papai

Landing page de presente de aniversário — um guia de tênis para viagem ao Japão, feito para facilitar a vida de quem vai comprar o presente.

## Sobre

Arquivo HTML único, sem dependências externas. A imagem está embutida em base64, então funciona offline e nunca quebra.

## Como usar

Abra o `index.html` em qualquer navegador. Pode ser enviado por WhatsApp, e-mail ou publicado no GitHub Pages.

## GitHub Pages

Se quiser publicar, ative o GitHub Pages nas configurações do repositório e acesse pelo link gerado.
